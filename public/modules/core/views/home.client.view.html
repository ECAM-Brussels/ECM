Exam Copies Manager

Welcome on the open-source solution to manage exams copies and to automatically generate pre-filled and ready-to-print PDF files for the exams.

Learn more

Management tool

ECM allows you to manage rooms, teachers and courses. Creating a new exam for a set of students and uploading its statement as a PDF file is made easy thanks to the intuitive interface.

Genuine algorithms

ECM contains algorithms for a genuine selection and attribution of rooms for the exams and for an automatic seat-assignment proposal of the students, with possible manual post-customisation.

PDF generation

ECM will automatically build ready-to-print and pre-filled PDF files, from a set of templates, with the statements of all the exams. Templates and fields to pre-fill are highly customisable.
